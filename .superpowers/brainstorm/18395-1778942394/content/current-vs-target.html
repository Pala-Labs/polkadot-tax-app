Key differences at a glance:
  • Color: Pink (#E6007A) → Blue (#2861E6). Single brand accent, used sparingly.
  • Radius: 16–20px rounded → 4px sharp corners throughout.
  • Cards: Frosted glass / backdrop-blur / drop shadow → flat white, 1px border, no shadow.
  • Buttons: Pill shape, scale-on-hover → 4px radius, translateY(1px) on press, no scale.
  • Type: Regular Inter → Inter Tight, tighter tracking, sentence case everywhere.
  • Icons/badges: Emoji + colored badges → mono code chips (JetBrains Mono), Lucide icons.
